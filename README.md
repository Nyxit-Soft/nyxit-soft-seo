# Nyxit SEO WordPress Plugin

## Features:

- All functionality is available as modules that can be easily activated/deactivated from the Settings > Nyxit SEO Settings menu.
- Meta information for posts and pages: meta descriptions, noindex, Open Graph, Twitter Card. Meta is added automatically in the head of every page. Open Graph and Twitter Card are completely optional.
- Breadcrumbs for posts and pages as an array. Structured data Breadcrumbs (json-ld) added to the post/page.  
- XML Sitemap for posts and pages according to Google's recommendations. Disables WP's build-in sitemap to avoid duplication.

## Usage:

Available shortcodes with their default values:

[nyxit_h1 id="" class=""]
